# JavaScript Alura

**Dinamismo** e **Interatividade**

#### **DOM** = Representação do mundo HTML no mundo JS

Toda pagina é representada pelo **document**

document.querySelector () = Utilizado para selecionar algum conteudo da pagina, como os elementos HTML por exemplo.

.textContent = Utilizado para trabalhar com o texto do elemento selecionado pelo querySelector.

.toFixed(). = Utilizado para Arredondar numeros decimais.

**Exemplo de sintaxe de uma função anonima :**
titulo.addEventListener("click", function(){
    console.log("Olá, eu posso chamar uma função anônima!");
})

**event.preventDefault();** = Evento que possui a função de previnir o comportamento padrão de um elemento, como o botão por exemplo.

**innerHTML** = Possui a função de controlar o html dentro de um elemento.
Ex:
**ul.innerHTML = "";**
No exemplo a cima o innerHtml possui a função de limpar o código sempre que for exibida uma mensagem de erro!

**forEach()** = Nessa função fazemos o que quisermos para cada item do array.

**this** = termo atrelado ao evento utilizado dentro do algoritmo js.
